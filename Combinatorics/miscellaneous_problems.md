### Problems

Given a string S print the letter with greatest number of occurrences, if there are two letters with the same number of occurrences return the one that occurs first in the string:


**input format:**

String S



**Examples:**


**sample input 1:**

S = "ABSCAA"

**sample output: A**


**sample input 2:**

S = "CDBRBSB"

**sample output 2: B**


**sample output 3:**

S = "ABC"

sample output3: NONE
